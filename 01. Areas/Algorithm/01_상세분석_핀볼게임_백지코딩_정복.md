# 🎳 실전 상세 분석: 5650. 핀볼 게임 (백지 코딩 편)

이 노트는 핀볼 게임의 전체 코드를 **"왜 이렇게 짰는지"** 아주 상세히 분석하여, 여러분이 빈 화면에서 이 코드를 직접 작성할 수 있게 돕습니다.

---

## 1. [핵심 로직] 방향 전환 테이블 (The change_dir Table)
핀볼의 핵심은 5개 블록마다 다르게 꺾이는 방향입니다. `if-else` 대신 리스트를 사용하면 코드가 5배 간결해집니다.

```python
# 방향 정의: 상(0), 하(1), 좌(2), 우(3)
# 각 블록(1~5)에서 특정 방향으로 들어왔을 때 나가는 방향을 미리 적어둡니다.
change_dir = [
    [], # 0번 (빈 공간)
    [1, 3, 0, 2], # 1번: 상(0)->하(1), 하(1)->우(3), 좌(2)->상(0), 우(3)->좌(2)
    [3, 0, 1, 2], # 2번: 상(0)->우(3), 하(1)->상(0), 좌(2)->하(1), 우(3)->좌(2)
    [2, 0, 3, 1], # 3번: 상(0)->좌(2), 하(1)->상(0), 좌(2)->우(3), 우(3)->하(1)
    [1, 2, 3, 0], # 4번: 상(0)->하(1), 하(1)->좌(2), 좌(2)->우(3), 우(3)->상(0)
    [1, 0, 3, 2]  # 5번: 모든 방향 반전 (상<->하, 좌<->우)
]
```
> [!caution] **백지 코딩 주의사항**
> 이 테이블을 짤 때 하나라도 틀리면 답이 안 나옵니다. 종이에 십자가를 그려보고, 각 블록의 대각선 모양을 확인하며 하나씩 채우는 습관을 들이세요.

---

## 2. [데이터 준비] 웜홀(Wormhole)의 사전 등록
매번 웜홀을 찾으러 격자를 뒤지는 행위는 삼성 시험에서 '탈락'의 지름길입니다.

```python
wormholes = {}
for r in range(N):
    for c in range(N):
        val = board[r][c]
        if val >= 6: # 웜홀은 6번부터 10번까지
            if val not in wormholes:
                wormholes[val] = []
            wormholes[val].append((r, c))
```
이렇게 해두면 `wormholes[6]`을 호출하는 것만으로 두 개의 구멍 좌표를 즉시 얻을 수 있습니다.

---

## 3. [시뮬레이션] 무한 루프 구현 (`while True`)
구슬은 언제 멈출까요? 딱 두 가지 상황입니다.
1. **블랙홀(-1)**을 만났을 때
2. **시작 위치**(`r, c`)로 돌아왔을 때

```python
while True:
    # 1. 일단 가던 방향으로 전진
    nr = curr_r + dr[curr_d]
    nc = curr_c + dc[curr_d]

    # 2. 벽에 부딪히는 경우 (가장 먼저 체크!)
    if not (0 <= nr < N and 0 <= nc < N):
        score += 1
        curr_d = reverse_dir(curr_d) # 방향 반전
        curr_r, curr_c = nr, nc # '벽 밖'의 좌표를 현재 좌표로 설정 (중요)
        continue

    # 3. 종료 조건 체크
    if (nr == start_r and nc == start_c) or board[nr][nc] == -1:
        # 최대 점수 갱신 후 루프 탈출
        max_score = max(max_score, score)
        break

    # 4. 블록을 만난 경우
    if 1 <= board[nr][nc] <= 5:
        score += 1
        curr_d = change_dir[board[nr][nc]][curr_d] # 테이블에서 즉시 방향 획득
        curr_r, curr_c = nr, nc

    # 5. 웜홀을 만난 경우
    elif board[nr][nc] >= 6:
        num = board[nr][nc]
        # 현재 들어간 좌표가 아닌 '다른' 좌표로 순간이동
        if wormholes[num][0] == (nr, nc):
            curr_r, curr_c = wormholes[num][1]
        else:
            curr_r, curr_c = wormholes[num][0]

    # 6. 그냥 빈 공간인 경우
    else:
        curr_r, curr_c = nr, nc
```

---

## 4. [A형 필수 최적화] "왜 모든 방향을 다 해봐야 할까?"
빈 공간(`0`)이라면 어느 방향으로든 쏠 수 있습니다. 
- 시작점 `(r, c)`가 `board[r][c] == 0`인 모든 곳을 찾아야 합니다.
- 각 지점에서 `상, 하, 좌, 우(4가지)` 방향으로 각각 구슬을 쏴봐야 합니다.
- 결과적으로 `N * N * 4` 번의 시뮬레이션이 일어나며, 이는 삼성 시험의 시간 제한 내에 충분히 들어옵니다.
