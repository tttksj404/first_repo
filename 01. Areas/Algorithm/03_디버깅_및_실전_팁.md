# 🩺 실전 디버깅 및 삼성 A형 취득 꿀팁 (IM → A형 완벽 마스터)

A형 시험은 **"10개 중 8개만 맞는 코드"**를 **"10개 다 맞는 코드"**로 고치는 싸움입니다. 실전에서 멘탈을 잡고 정답을 찾아내는 팁을 5배 더 상세히 정리했습니다.

---

## 1. [자가 진단] "내 코드는 왜 80%만 맞을까?"
삼성 시험에서 80%만 맞는다면, 다음 4가지 중 하나가 범인입니다.

### 1-1. 2차원 배열의 얕은 복사 (The Copy Trap)
```python
# ❌ 절대 안 됨 (원본이 같이 바뀜)
temp_board = board 

# ❌ 삼성 시험에서는 지양 (매우 느림)
import copy
temp_board = copy.deepcopy(board)

# ✅ 가장 빠르고 안전한 방법 (슬라이싱)
temp_board = [row[:] for row in board]
```

### 1-2. 방향(dr, dc)의 순서 불일치
- 문제: `1:상, 2:하, 3:좌, 4:우`
- 내 코드: `dr = [-1, 1, 0, 0], dc = [0, 0, -1, 1]`
- **팁**: 인덱스를 맞추기 위해 `dr = [0, -1, 1, 0, 0]` 처럼 앞에 더미(0)를 넣으세요. 그러면 `dr[1]`이 상, `dr[2]`가 하가 되어 실수가 줄어듭니다.

## 2. [가상화] 격자 상태 예쁘게 출력하기 (`print_board`)
눈으로 격자를 보지 않으면 웜홀 위치나 상어 이동의 오류를 찾을 수 없습니다.

```python
def print_board(board, msg=""):
    print(f"--- {msg} ---")
    for row in board:
        # 가독성을 위해 숫자를 칸 맞춰서 출력
        print(" ".join(f"{x:3}" for x in row))
    print("-" * 20)

# 실전 사용법
# 1라운드 상어 이동 후 호출
move_sharks()
print_board(board, "1ROUND 이동 후")
```
> [!tip] **실전 팁**
> 제출할 때는 반드시 이 함수 호출을 지우거나 주석 처리하세요. 출력량이 너무 많으면 시간 초과가 납니다.
