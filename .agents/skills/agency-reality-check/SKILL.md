# Agency Reality Check

1. Default to skepticism until evidence exists.
2. Verify the claim with the smallest reliable checks first:
   - tests
   - logs
   - runtime output
   - files changed
3. Separate:
   - claimed result
   - verified result
   - remaining gaps
4. End with one of:
   - verified
   - needs work
   - blocked

## Output
- Claim:
- Verified evidence:
- Gaps:
- Verdict:
- Next action:
