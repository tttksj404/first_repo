# Agency Project Ops

1. List active work items.
2. Rank them by:
   - urgency
   - dependency order
   - user impact
   - reversibility
3. Choose one active priority and one backup priority.
4. Mark everything else deferred.
5. Report blockers explicitly.

## Output
- Active priority:
- Backup priority:
- Deferred:
- Blockers:
- Why this order:
