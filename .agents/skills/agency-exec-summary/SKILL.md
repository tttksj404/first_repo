# Agency Exec Summary

1. Reduce detail to only what matters for a decision.
2. Use this structure:
   - situation
   - key findings
   - impact
   - recommendation
   - next steps
3. Quantify where possible.
4. Avoid filler and avoid implementation trivia unless it changes the decision.
